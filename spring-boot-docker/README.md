# docker command

### docker image build

```
docker build -t image-name .
```

### docker run

```
docker run -p [host]:[container] image-name
```

### docker images

```
docker images
```

### docker process

```
docker ps
docker ps -a
```

### delete docker container

```
docker rm container-id
```

### delete docker image

```
docker rmi image-id
```
